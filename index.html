Note

  • thuộc tính for của thẻ < label /> phải giống với id của thẻ < input /> tương ứng
  • < input type="checkbox" /> lựa chọn được nhiều -> khác name
  • < input type="radio" /> thuộc tính name phải trùng nhau nếu muốn lựa chọn 1 trong những giá trị và phải có thuộc tính value để phân biệt
  • thuộc tính require trong form giúp bắt buộc điền giá trị trước khi gửi
